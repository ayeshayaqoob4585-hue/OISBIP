# Iris Flower Classification

## Oasis Infobyte Internship - Task 1

### Project Description
This project classifies Iris flowers into three species using Machine Learning. The Decision Tree Classifier is used for training and prediction.

## Dataset
- Iris Dataset (Scikit-learn)

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Features
- Data Loading
- Data Exploration
- Pairplot Visualization
- Correlation Heatmap
- Decision Tree Classification
- Confusion Matrix
- Classification Report
- Accuracy Score
